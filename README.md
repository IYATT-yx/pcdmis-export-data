# pcdmis-export-data

PC-DMIS 测量数据自动化导出工具。将它添加到测量程序中，随测量程序调用执行后会读取已测量的数据保存到 Excel 文件中，并对超差值进行标识。超上差的值填充为红色，超下差的值填充为品红，所有检测项合格，序列号填充绿色，否则填充黄色。
